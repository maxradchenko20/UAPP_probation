font-size:

Нормальный стиль шрифта. Значение по умолчанию //default
Курсивный стиль шрифта // italic
Косой (наклонный) стиль шрифта // oblique
Значение должно быть унаследовано от элемента родителя - default // inherit

font-style:

Нормальный стиль шрифта. Значение по умолчанию //default
Курсивный стиль шрифта // italic
Косой (наклонный) стиль шрифта // oblique
Значение должно быть унаследовано от элемента родителя - default // inherit

font-variant:

Нормальный шрифт. Значение по умолчанию //normal
Все буквы становятся прописными // small-caps
Значение должно быть унаследовано от элемента родителя // inherit

font-weight:

Нормальный шрифт. Значение по умолчанию //normal
Шрифт должен быть тоньше (менее насыщеннее) по отношению к элементу родителю // lighter
Толстый шрифт// bold
Шрифт должен быть толще (более насыщеннее) по отношению к элементу родителю // bolder
Значение должно быть унаследовано от элемента родителя // inherit
